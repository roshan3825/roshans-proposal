# Roshan's Proposal Website

A multi-page romantic interactive proposal website for GitHub Pages.

## Pages
- `index.html` - proposal question, with moving No button
- `cuisine.html` - cuisine selection
- `availability.html` - date and time selection
- `review.html` - contact number and Formspree submission
- `success.html` - confirmation page

## Formspree setup
1. Go to Formspree and create a form.
2. Copy your endpoint, for example `https://formspree.io/f/abcdwxyz`.
3. Open `config.js`.
4. Replace `https://formspree.io/f/YOUR_FORM_ID` with your endpoint.
5. Upload the whole folder to GitHub Pages.
6. Test the form. First submission may require email confirmation inside Formspree.

## Important
Do not remove the `assets` folder. It contains all CSS, JavaScript, and chibi character images.
